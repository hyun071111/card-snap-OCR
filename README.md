### 환경설정
macOS 또는 Linux
```
# On macOS and Linux.
curl -LsSf https://astral.sh/uv/install.sh | sh
또는 
brew install uv
```
windows
```aiignore
# On Windows.
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
PyPI로 설치
```
# With pip.
pip install uv

# Or pipx.
pipx install uv
```
